What I wanted for so long is now here: client-side encryption on a web page with Yubikey (or platform passkey, or password):
and I called that **Texcrets**!

TL;DR; I can write posts here with blocks like:

```
{ % texcret %}
This text must be hidden from public eyes (read encrypted).
{ % endtexcret %}
```

_(I put extra space in tag opennings to go under the radar with the example above)._

And they are *texcreted* into blocks like this at pre-commit via a *texcret* cli:

Which I can temporarily decrypt (in-page without any network communication)
magically by performing the following click pattern on it: *dblclick - click - dblclick*.

Check the [demo](https://aleksandr.vin/texcret/demo.html) or github [repo](https://github.com/aleksandr-vin/texcret).
